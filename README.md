
# L&M — Statický web (jednoduchá varianta)

Jednoduchý a rychlý web bez build nástrojů (stačí HTML/CSS/JS). Ideální na Vercel nebo GitHub Pages.

## Struktura
- `/images/` — sem nahraj fotky (viz názvy níže)
- `index.html`
- `styles.css`
- `script.js`
- `logo.svg`

## Jak spustit lokálně
Stačí otevřít `index.html` v prohlížeči (dvaklik).

## Nasazení na Vercel
1. Vytvoř nový repo na GitHubu (např. `l-m-web`).
2. Nahraj soubory z tohoto ZIPu do kořene repozitáře.
3. Na Vercel klikni **Add New → Project → Import z GitHubu**.
4. Framework: **Other**, Root Directory: `./`. Deploy.

## GitHub Pages
`Settings → Pages → Build and deployment → Deploy from a branch`, branch `main`, folder `/ (root)`.

## Výměna fotek
V `/images` nahraj fotky pod těmito názvy:
`hero.jpg`, `gal-1.jpg`, `gal-2.jpg`, `gal-3.jpg`, `gal-4.jpg` (doporučeno 1600×1200 JPG).

## Texty/Barvy
Úpravy v `index.html` a `styles.css`.
